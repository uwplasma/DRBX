# Field-Aligned 3D Example

This example runs a field-aligned s-alpha configuration and produces 3D slice
plots and a midplane movie.

## Run

```bash
python examples/field_aligned_3d/run.py --make-figures --make-movies
```

Outputs:
- `examples/field_aligned_3d/output.npz`
- `docs/figures/three_d_slices.png`
- `docs/figures/three_d_rms_timeseries.png`
- `docs/figures/three_d_movie.gif`
