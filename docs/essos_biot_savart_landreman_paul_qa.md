# ESSOS Biot-Savart Landreman-Paul QA Campaign

This page documents the first coil-produced magnetic-field lane in `jax_drb`.
It adds an ESSOS-compatible Fourier-coil loader, a JAX-native Biot-Savart
field evaluator, and a compact annular field-line-map turbulence campaign for
the `ESSOS_biot_savart_LandremanPaulQA.json` coil set.

The campaign is regenerated by:

```bash
JAX_DRB_ESSOS_ROOT=/path/to/ESSOS \
PYTHONPATH=src .venv/bin/python examples/geometry-3D/essos-biot-savart/landreman_paul_qa_campaign.py
```

If `--coil-json` is passed, the example uses that JSON file directly. The
committed report stores only the coil filename and format metadata so public
docs do not depend on a local checkout path.

## Scope

This is a coil-field and FCI-operator validation gate, not a wall-resolved
stellarator edge prediction. It proves that `jax_drb` can ingest an
ESSOS-format Fourier coil file, reconstruct the symmetry-expanded filament
set, evaluate the magnetic field with JAX, build closed-like and open/SOL-like
annular FCI maps from that field, and run a bounded reduced scalar turbulence
response on both regions. Full research closure still requires imported
plasma-boundary and wall geometry, target masks, sheath-normal projection,
neutral recycling deposition, vorticity/potential coupling, and the production
fixed-layout DRB RHS.

## Coil And Field Equations

Each base coil is represented by Cartesian Fourier coefficients. With
\(\tau \in [0,1)\),

```text
gamma_c(tau) = a_c0
             + sum_m [a_c,m^sin sin(2 pi m tau)
                      + a_c,m^cos cos(2 pi m tau)]

gamma'_c(tau) = sum_m 2 pi m [a_c,m^sin cos(2 pi m tau)
                              - a_c,m^cos sin(2 pi m tau)]
```

The loader applies field-period rotations and stellarator-symmetry reflection
using the same current-sign convention as the source coil file. The magnetic
field is then evaluated by the filament Biot-Savart quadrature

```text
B(x) = mu0 / (4 pi) sum_c I_c integral
       gamma'_c(tau) cross [x - gamma_c(tau)]
       / |x - gamma_c(tau)|^3 d tau
```

and the discrete implementation uses the segment average

```text
B(x) ~= 1e-7 mean_tau sum_c I_c
        gamma'_c(tau) cross [x - gamma_c(tau)]
        / max(|x - gamma_c(tau)|^3, epsilon^3)
```

The focused unit gate checks this formula against the analytic on-axis field
of a circular current loop,

```text
B_z(0) = mu0 I / (2 R),
```

and also checks that `jax.jvp` can differentiate the field with respect to the
observation point.

## Annular FCI Maps

The campaign samples two annular regions around a coil-derived axis estimate.
For a point

```text
R = R0 + rho cos(theta)
Z = Z0 + rho sin(theta)
x = R cos(phi)
y = R sin(phi),
```

the Cartesian field is projected onto cylindrical components
\((B_R, B_phi, B_Z)\). A one-plane field-line step in toroidal angle is
constructed from

```text
dR / dphi = R B_R / B_phi
dZ / dphi = R B_Z / B_phi.
```

The resulting \((R,Z)\) point is mapped back to annular indices
\((rho, theta)\). If the mapped point leaves the annulus, the FCI map records
a boundary hit. In the current Landreman-Paul QA coil campaign, the closed-like
inner annulus has boundary-hit fraction about `0.413`, while the outer
open/SOL-like annulus has boundary-hit fraction about `0.681`. This difference
is the gate that the outer annulus samples a more open-field-line-like region
than the inner annulus.

## Reduced Dynamics

Both annuli evolve the same compact scalar fluctuation model:

```text
partial_t n_tilde
  = chi_parallel L_parallel^FCI(n_tilde)
  + chi_perp L_perp(n_tilde)
  + S(rho, theta, phi, t)
  + alpha C_B partial_rho n_tilde
  - beta n_tilde
  - beta_target M_endpoint n_tilde
  - gamma n_tilde^3
  - n_tilde partial_theta n_tilde.
```

Here \(L_parallel^FCI\) is the traced field-line-map diffusion operator,
\(L_perp\) is a compact radial-poloidal diffusion operator, \(S\) is a
time-dependent seed/source pattern, \(C_B\) is the normalized radial-field
fraction used as a curvature/interchange proxy, and \(M_endpoint\) is the
forward/backward endpoint mask. The open/SOL-like annulus uses a stronger
endpoint loss to make the contrast with the closed-like annulus visible in a
short documentation run.

The current report passes with `16` symmetry-expanded coils, `75` coil
segments, two field periods, and stellarator symmetry enabled. The closed-like
annulus has final RMS fluctuation about `7.07e-2`; the open/SOL-like annulus
has final RMS fluctuation about `5.96e-2`. A separate annular field-line trace
now verifies that the inner annulus has a longer residence-length proxy than
the outer annulus: the mean annular exit time is about `0.381` toroidal turns
inside and about `0.155` toroidal turns outside, while the mean exit
connection-length proxy is about `3.11` inside and `1.77` outside. Those values
are intentionally reported as reduced geometry/dynamics diagnostics rather than
as physical turbulence predictions or wall strike-point predictions.

## Next Implementation Steps

The coil-field lane should now be promoted in the same order as the analytic
stellarator FCI lane:

1. Replace the annular axis estimate with a traced magnetic-axis or imported
   surface/wall bundle.
2. Promote the annular Poincare and residence diagnostics to imported
   surface/wall bundles with real closed, island, stochastic, and open-field
   region labels.
3. Add conservative metric-weighted FCI diffusion using the coil-produced
   field and imported geometry metrics.
4. Route sheath, target recycling, and neutral sources through the production
   fixed-layout RHS instead of the reduced scalar demo.
5. Add vorticity/potential coupling so radial particle and heat fluxes are
   computed from \(E \times B\) dynamics rather than a scalar proxy.
6. Add batched source and coil-parameter scans with `jax.vmap`, `jax.jvp`, and
   optional multi-device execution after parity gates are clean.
