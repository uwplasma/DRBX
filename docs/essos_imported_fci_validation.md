# ESSOS Imported FCI Validation

This page documents the first downstream use of externally traced
Landreman-Paul QA field lines inside `jax_drb` FCI operators. ESSOS supplies
the coil-field evaluation and adaptive trajectories. `jax_drb` converts those
trajectories into fixed-shape plane-to-plane maps, builds a lightweight
annular metric for the imported logical grid, and then evaluates JAX-native
sheath/recycling and neutral reaction-diffusion closures on those maps.

The campaign is regenerated by:

```bash
JAX_DRB_ESSOS_ROOT=/path/to/ESSOS \
PYTHONPATH=src .venv/bin/python \
  examples/geometry-3D/essos-field-lines/imported_fci_campaign.py
```

The example accepts `--coil-json` or `--essos-root` when the ESSOS checkout
is not located at the default `~/local/ESSOS`.

## Geometry Import

The imported grid is an annulus centered on the magnetic axis reported by the
external Biot-Savart field object. Seeds are placed at
\((\rho,\phi,\theta)\) and converted to Cartesian coordinates through

\[
R = R_0 + \rho\cos\theta,\qquad
Z = Z_0 + \rho\sin\theta,\qquad
\mathbf{x}=(R\cos\phi,R\sin\phi,Z).
\]

Forward and backward trajectories are traced from every seed. For each seed,
the adapter interpolates the external trajectory to the adjacent toroidal
planes \(\phi\pm\Delta\phi\), converts the endpoints back to logical
\((x,z)\) coordinates, and marks a boundary if the endpoint leaves the
annulus or does not reach the adjacent plane. Boundary map indices are stored
as finite placeholders and the boundary mask carries the physics meaning; this
keeps the JAX interpolation kernels shape-stable and safe under `jit`, `vmap`,
`jvp`, and future implicit residual promotion.

The annular metric used by this first imported-map gate is the cylindrical
logical metric

\[
J = R\rho,\qquad
g^{\rho\rho}=1,\qquad
g^{\phi\phi}=R^{-2},\qquad
g^{\theta\theta}=\rho^{-2},
\]

with cross terms set to zero. This is intentionally a validation bridge: the
field-line maps come from the external coil-field tracer, while the metric is
simple enough to make conservation and closure accounting directly
interpretable.

## Physics Gates

The sheath/recycling gate applies a normalized Bohm target flux to every
forward or backward field-line endpoint,

\[
\Gamma_i = N_i\sqrt{(T_e+T_i)/m_i},
\]

reconstructs the electron particle flux from zero-current balance, and checks
that recycled particle and neutral-energy sources exactly close their global
accounting identities. The neutral gate then evaluates FCI parallel diffusion,
perpendicular metric diffusion, ionisation, recombination, and charge exchange
on the same imported maps. The report records endpoint fractions, magnetic
field modulation, connection-length statistics, target heat-load contrast,
particle balance residuals, current residuals, and neutral momentum balance.
