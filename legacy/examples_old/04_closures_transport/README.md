# Transport / closures

Scripts focused on parallel closures and Braginskii/Spitzer transport scalings.

- `parallel_closures_effects.py`: constant-coefficient parallel conduction/diffusion knobs.
- `braginskii_closures_effects.py`: equilibrium-based Braginskii/Spitzer scalings for
  $\eta(T_{e0})$, $\chi_\parallel(T_0)$, and $\nu_\parallel(T_0)$.
- `nonlinear_flux_tube_toggles.py`: short nonlinear field-line run comparing
  equilibrium-based vs state-dependent non-Boussinesq polarization and Braginskii coefficients.
